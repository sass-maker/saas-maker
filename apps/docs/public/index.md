# SaaS Maker API Documentation

Drop-in backend services for SaaS apps. Base API: https://api.sassmaker.com

## Auth

- Project API key via `X-Project-Key` for public SDK/widget endpoints
- Session/CLI Bearer tokens for dashboard and admin operations
