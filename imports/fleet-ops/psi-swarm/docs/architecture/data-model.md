# Data model

All local state lives in a single SQLite database at
`~/.psi-swarm/history.db`, opened by `cli/src/db.ts` (`HistoryDB`). The schema
is created idempotently on first open (`CREATE TABLE IF NOT EXISTS`). There
are no migrations files — schema evolution is handled inline (see the
`migrateDomainRatingsNullable` note below).

## Tables

### `runs`

One row per (URL × preset × single Lighthouse execution). The unit of
history.

| Column | Type | Notes |
| --- | --- | --- |
| `id` | INTEGER PK AUTOINCREMENT | |
| `url` | TEXT NOT NULL | The audited URL. |
| `preset` | TEXT NOT NULL | e.g. `mobile-mid`, `desktop`. |
| `started_at` | INTEGER NOT NULL | Epoch ms. |
| `finished_at` | INTEGER | Null while in flight. |
| `lcp`, `cls`, `inp`, `tbt`, `fcp`, `ttfb`, `si` | REAL | Web Vitals / timing metrics. May be NULL (e.g. INP absent in lab). |
| `performance_score` | REAL | Lighthouse perf score. |
| `error` | TEXT | Populated when the run failed (batch runs continue on error). |
| `tag` | TEXT | Optional `--tag` for `compare` (e.g. `before-deploy`). |

Indexes: `(url, preset)`, `started_at`, `tag`.

### `domain_ratings`

Ahrefs Domain Rating cache for custom-domain projects.

| Column | Type | Notes |
| --- | --- | --- |
| `domain` | TEXT PK | The hostname. |
| `rating` | REAL | **Nullable** — a NULL row negative-caches "Ahrefs has no rating". |
| `fetched_at` | INTEGER NOT NULL | Epoch ms. |

> **Migration note.** Older DBs created this column as `REAL NOT NULL`. It
> was made nullable so a NULL row can negative-cache a no-rating result.
> `migrateDomainRatingsNullable()` in `db.ts` patches legacy DBs in place
> by inspecting `table_info`.

Skipped entirely for `*.pages.dev` / `*.workers.dev` (DR on shared CF
subdomains is not meaningful). Refreshed weekly while idle by the
[domain-rating scheduler](../operations/background-jobs.md).

### `run_insights`

Derived trace-insight diagnosis, one row per saved swarm's run.

| Column | Type | Notes |
| --- | --- | --- |
| `run_id` | INTEGER PK | FK → `runs(id)` ON DELETE CASCADE. |
| `bottleneck_phase` | TEXT | Dominant LCP phase. |
| `summary` | TEXT NOT NULL | One-line diagnosis. |
| `opportunities_json` | TEXT NOT NULL DEFAULT '[]' | Ranked opportunities array. |
| `comparison_notes` | TEXT | Optional baseline delta notes (`--insight-baseline`). |
| `adapter` | TEXT NOT NULL | `builtin` or the external adapter id. |
| `artifact_path` | TEXT | Path under `~/.psi-swarm/artifacts/`. |
| `created_at` | INTEGER NOT NULL | Epoch ms. |

See the [trace-insight PRD](../prds/trace-insight-adapter.md).

### `watchlist`

The local regression watchlist.

| Column | Type | Notes |
| --- | --- | --- |
| `url` | TEXT PK | The watched URL. |
| `label` | TEXT | Optional human label. |
| `preset` | TEXT NOT NULL DEFAULT 'mobile-mid' | Which preset to evaluate against. |
| `baseline_tag` | TEXT | Optional baseline tag for comparison. |
| `lcp_threshold_ms` | REAL | Regression threshold; default +200 ms or +10%. |
| `score_threshold` | REAL | Default perf score −5 points. |
| `stale_days` | INTEGER NOT NULL DEFAULT 7 | Stale-after window. |
| `added_at` | INTEGER NOT NULL | Epoch ms. |

Index: `added_at`. See the [watchlist PRD](../prds/local-regression-watchlist.md).

### `meta`

Generic key/value store. Currently holds `ahrefs_last_refresh_at` (used by
the domain-rating scheduler to throttle weekly refreshes).

| Column | Type |
| --- | --- |
| `key` | TEXT PK |
| `value` | TEXT NOT NULL |

## On-disk layout

```
~/.psi-swarm/
├── history.db                  ← the database above
├── artifacts/<swarm-id>/       ← exported Lighthouse bundles (one .json per preset)
├── adapters/trace-insight.mjs  ← optional external adapter hook
└── reports/                    ← generated --output html reports
```

The report registry in `cli/src/server.ts` also scans `/tmp/psi-*` dirs so
reports generated by fleet batch scripts are discoverable via `/api/reports`.
