# HeyPace

**On-device Mac voice agents.**

HeyPace builds [Pace](https://heypace.app) — an on-device Mac voice agent that
reads your screen and acts with local context. No cloud round-trip, no data
leaving your machine. Pace observes, reads, and acts where you work.

## What we ship

- **[Pace](https://heypace.app)** — on-device Mac voice agent. Screen-aware assistance with local context. $29 one-time purchase.
