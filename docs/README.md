# SaaS Maker documentation

This tree is the source for packages.sassmaker.com. Blume is the only
presentation layer.

The public documentation covers @saas-maker/feedback, its project keys, and its
small HTTP API. Fleet operations, marketing automation, App Health, CodeVetter,
skills, and schedules belong to Fleet Workspace or their independent products
and must not be documented here.

Start with [the quickstart](getting-started/quickstart.md).
