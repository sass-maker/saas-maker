# Research

Research notes that informed architecture decisions. These are not decisions
themselves — for those, see [`../decisions/`](../decisions/). Research notes
capture options surveyed, trade-offs, and external references at a point in
time.

## Files

- [`agent-operating-systems-openroom-openclaw-dynamic-workers.md`](agent-operating-systems-openroom-openclaw-dynamic-workers.md) —
  layered model of agent operating systems (OpenRoom, OpenClaw, Dynamic
  Workers) that informed the Dynamic Workers Symphony plan.
- [`free-database-options-2026.md`](free-database-options-2026.md) — survey of
  free-tier database options considered for the fleet.

## Conventions

- File names are `<topic>.md` or `<date>-<topic>.md`.
- Cite authoritative external sources; keep entries to "what", "why it
  matters to this project", and a link — do not re-explain material that has
  a definitive source.
- When research leads to a decision, link the decision record in
  [`../decisions/`](../decisions/) and vice versa.
