# Product Knowledge

What SaaS Maker is, who uses it, what is in and out of scope, and the fleet it
operates. For the public product surface (API, SDK, widgets, services), see
[`../getting-started/`](../getting-started/) and [`../services/`](../services/).

## Files

- [`fleet-registry.md`](fleet-registry.md) — canonical display names and
  classification for every active fleet project. `foundry.projects.json` is the
  machine-readable source of truth; this file is the human-readable contract.
- [`ideas.md`](ideas.md) — project-specific product thinking and module ideas
  (moved from `IDEA-DUMP.md`).
- [`saas-ideas-consolidation-2026-06-03.md`](saas-ideas-consolidation-2026-06-03.md) —
  consolidated SaaS idea notes.
- [`agent-company-stack.md`](agent-company-stack.md) — reference parts list for
  an "AI builds + operates a company" platform (research reference).
- [`project-recommendation-context.md`](project-recommendation-context.md) —
  context used when recommending projects for fleet attention.
