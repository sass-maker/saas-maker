# Creator MVP Packs

These packets are the manual production source of truth for the first kids
story validation videos.

They are intentionally not automation. Each packet should let a creator make
one complete video with a public-domain source note, rewritten narration, scene
list, image prompts, voice direction, edit notes, thumbnail/title options, and
final upload checklist.

## Production Order

1. `lion-and-mouse.md`
2. `tortoise-and-hare.md`
3. `crow-and-pitcher.md`

Do not add render automation, schedulers, auto-uploaders, dashboards, or
analytics scripts for this kids-story bet until these three videos exist and
have been reviewed against the parent-trust checklist in `../creator-mvp.md`.

## Shared Visual Style

Use the same style across all three videos:

> Warm children's picture book illustration, soft watercolor texture, gentle
> natural light, friendly expressive animals, calm composition, no text in the
> image, 16:9.

Keep character designs consistent within each story. If an image tool supports
seed/reference locking, reuse it for the whole packet.

## Shared Edit Style

- 5-8 minutes.
- 10-14 scenes.
- Slow image movement only: soft fades, slow pan, slow zoom.
- Warm narration at bedtime pace.
- Soft music and light sound effects.
- No loud stingers, jump cuts, shaky camera, meme edits, or fast Shorts pacing.

## Validation Gate

After all three videos are made, record:

- average view duration;
- where viewers drop off;
- one parent-trust note per video;
- whether the next iteration needs better scripts, better voice, better art, or
  only then a small software assist.
